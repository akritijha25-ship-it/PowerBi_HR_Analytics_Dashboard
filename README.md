# PowerBi_HR_Analytics_Dashboard

## Overview
This dashboard focuses on employee attrition analysis.
It helps identify patterns related to age, gender, education,
salary slabs, job roles, and years at company.

## Tools Used
- Power BI Desktop
- Power Query
- DAX

## Key Insights
- Overall attrition rate and employee statistics
- Attrition by age group, education, and gender
- Impact of salary slabs on attrition
- Job role and job satisfaction analysis
